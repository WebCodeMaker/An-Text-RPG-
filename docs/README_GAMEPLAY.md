# Gameplay Reference

## Core Loop
1. Explore zones to trigger encounters.
2. Win battles to gain XP and gold.
3. Spend gold in the shop to improve survivability.
4. Complete quests for large reward spikes.
5. Rest periodically to reset health and mana.

## Combat Depth
- **Basic Attack** scales with strength + weapon power.
- **Fireball** can apply Burn (damage over time).
- **Heal** scales with max HP, so it improves naturally with progression.
- **Armor** reduces incoming damage each enemy turn.

## Progression Strategy
- Prioritize an early weapon for reliable damage.
- Keep at least one Ether in inventory before difficult zones.
- Finish "Goblin Menace" as soon as possible for strong early rewards.

## Save/Load Notes
- Save anytime from the main menu.
- Save file path: `savegame.json` in repository root.
- Loading preserves level, stats, inventory, equipment, and quest progress.
